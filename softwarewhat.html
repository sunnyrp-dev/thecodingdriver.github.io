What is Software?


  • A Software is a set of instructions and collection of programs, data, and instructions that tell a computer or electronic device what to do and how to do it.
  • Software is the brain that runs the physical machine (Hardware).
  • Software is the brain of a computer — it controls hardware and makes tasks possible.
  • Ex- Media Player, PDF Reader


Types of Software


  1. System Software
    • Runs and controls the computer hardware.
    • Example: Windows, Linux, macOS

  2. Application Software
    • Used by users to perform tasks.
    • Example: MS Word, Chrome, Photoshop

  3. Programming Software
    • Used by developers to create applications.
    • Example: VS Code, Java, Python

  4. Embedded Software
    • Runs inside devices and machines.
    • Example: Software in cars, washing machines, smart TVs
